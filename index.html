Tech Used: MySQL - FLASK - PYTHON - HTML/JINJA

Pickleball Finder App - Full Stack Project

Personal Project Project

Pickleball Finder is a web application developed using Node.js with RESTful routing capabilities. Its primary objective is to assist users in discovering nearby pickleball courts. The application allows users to create accounts, log in, post reviews, and suggest new pickleball courts. Furthermore, the application uses the Cloudinary and MapBox API's to manage image storage and map-related features.


Tech Used: Javascript - Express - MongoDB - HTML/CSS

Features: Authentication - Authorization - Flash Messages - Functionalities of court posts and comments - Responsive web design
